安卓电视安装KODI

发表于 2023-07-30 分类于 KODI

文件传输到电视方式：

  1. 电视安装【小白文件管理器】，使用其中的【多屏互动-快速传输】，来实现电脑传送文件到电视。
  2. 如果没有【小白文件管理器】，如果有【悟空遥控器】，可以安装【悟空遥控器】后在该软件应用市场安装【小白文件管理器】
  3. U盘。

socks5代理协议服务端搭建

发表于 2023-07-17 更新于 2023-07-21 分类于 socks5

所用项目：https://github.com/Lozy/danted/

安装和管理方式在GitHub项目主页都有，这里不再重复。
需要注意的是，如果机器有多个IP，且配置文件中无 external.rotation: same-same 配置，则要手动加上。

配置文件：/etc/danted/sockd.conf
在配置文件 clientmethod: none 上面加一行：
external.rotation: same-same

把 OPPOSans-R.ttf 和 Arial-Regular.ttf 放到项目根目录的fonts目录内，执行：
php ./load_font.php ‘OPPOSans-R’ fonts/OPPOSans-R.ttf

haproxy配置TCP多目标流量转发

发表于 2023-05-03 更新于 2023-05-04 分类于 Linux

apt install haproxy

cat /lib/systemd/system/haproxy.service
得到配置文件位置：/etc/haproxy/haproxy.cfg

vim /etc/haproxy/haproxy.cfg

systemctl start haproxy

访问 IP:880 端口，则服务器会代理客户端访问server1、2、3，并返回结果给客户端。

注意每次修改配置后要 systemctl restart haproxy 重载配置。
