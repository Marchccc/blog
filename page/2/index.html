网站接入海外第三方社交登录

发表于 2020-12-24 分类于 web

接入海外第三方登录，服务器需要与海外应用通信，服务器解决方式为：国内流量由国内服务器处理，国外流量由国外服务器处理，通过域名解析线路实现。
如果域名使用了CDN并且设置了全球加速，则需要把全球加速关闭，改为仅加速国内，否则国外DNS可能会解析到CDN，而CDN又回源到国内服务器了。
如果www前缀有设置CNAME，则需要再设置一条境外的。
国内访问 -> DNS解析 -> 国内服务器
海外访问 -> DNS解析 -> 海外服务器
